# Romain Wisniewski – Photographe

## À propos
Romain Wisniewski est un photographe français né le 30 octobre 2004 à Gien. Il s'intéresse à la photographie documentaire, d'abord en France, puis dans l'ensemble de l'Europe au travers de ses voyages.
